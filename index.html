Chocolates l'creme:

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma empresa de chocolates. O objetivo foi ter o primeiro contato com os sistemas HTML e CSS e entender suas estruturas e como funcionam.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.choco) que mostra as principais informações da empresa e seu produto.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores com tons de marrom. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos.

POPfilm:

O projeto consiste em uma empresa que está oferecendo um combo, desenvolvida em HTML e CSS. O objetivo era criar um site mais interativo com botões e aprnder a usar o âncora.

O HTML estrutura o conteúdo da página, como a imagem da logo, os botões, as informações sobre o combo, os lugares onde se pode acessar o streaming e o rodapé.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site.

Com o CSS estilizamos nosso site com cores que variam do preto para o cinza, ajustamos as imagens, adicionamos uma imagem como plano de fundo, demos cores aos botões que mudam de cor ao passarmos o mouse por cima, modificamos as cores e fontes dos textos.

Clalyn:

O projeto é uma livraria online com foco em livros de romance. O objetivo era criar um site onde existisse um pouco mais de interação, onde se tem botões no menu que mostra as categorias, botões para adicionar ao carrinho, marcar como favorito e ver perfil.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro é destacado em seu próprio contêiner (.livro), mostrando uma imagem da capa, detalhes do livro e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

Com CSS personalizamos o visual do site, usando uma paleta de cores mais romântica para combinar com o tema, estilizamos os textos com uma fonte importada do Google Fonts Moderustic e botões interativos.

Jornada EDUCATECH: Smart Bracelet For The Blind

Participamos da jornada com o projeto Smart Bracelet For The Blind, que é uma pulseira com o objetivo de ajudar pessoas com deficiência visual a atravessar ruas sinalizadas por semáforos.

que foncionaria da seguinte forma: quando a pessoa que estivesse utilizando a pulseira, se aproximasse do semáforo dos pedestres ela emitiria um som ou vibração quando ele estivesse verde.
